Extension to simplify delta 3d printer calibration process with Repetier by using Escher 3D

  1. Copy script to repetier host, check dimensions
  2. Run M205, copy console from Repetier Host and click EPROM to setup your printer
  3. Clean up console in repetier host
  4. Run script
  5. Copy console from repetier host
  6. Paste into page
  7. Click Calculate button
  8. Copy & paste commands
  9. Repeat process (don't forget to copy initial parameters) from 3
