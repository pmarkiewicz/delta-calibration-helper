Extension to simplify delta 3d printer calibration process by using Escher 3D

  1. Copy script to repetier host
  2. Check height and dimension and change for your printer
  3. Clean up console in repetier host
  4. Run script
  5. Set your printer parameters on page
  6. Copy console from repetier host
  7. Paste into page
  8. Click Calculate button
  9. Copy & paste commands
  10. Repeat process (don't forget to copy initial parameters)
